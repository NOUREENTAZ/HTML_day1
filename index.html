5. Difference between block and inline?

  1. Block occupy the full width irrespective of their sufficiency where as Inline occupy only sufficient width requried.
  2. Block always start with new line and Inline did not start with new line.
  3. Block have top and bottom margin. Inline did not have top and bottom margin.
  4. Block it will add the line break after the element so the element sit next line to other element Inline this element does not add a line break after the element, so the element can sit next to other elements.
  5. Block Ex- <p> , <div> , <h1> to <h6> , <form> , <table> .... Inline Ex- <span> , <a> , <img>, <input> , <select> ....

7. What are attributes?

HTML attributes are special words used inside the opening and closing tag to control the element's behaviour.

Attributes provide additional information about elements. usaually come in name/value pair like name="value".

Always lower case it's case sensitive

Syntax <element attribute_name="value"> content <element>

Ex- <a href="google.com" target="_blank">
where href and target are attribute for anchor tag.
